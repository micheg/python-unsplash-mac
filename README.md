# python-unsplash-mac
A simple script that change all desktops background via unsplash Api on Mac 

# Requirements:
* Mac OS
* Python

# Tested Version:
* OS Version  : macOS High Sierra 10.13.6
* Python Version  : 3.6.5

# Install
```bash
python setup.py
```

# Start
```bash
python main.py
```

#Alias
``` 
alias unsplash="python ~/Desktop/unsplash-python/main.py"
```
